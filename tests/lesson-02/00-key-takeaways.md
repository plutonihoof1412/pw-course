
## Git Locations
- WorkingDir: personal playground, can add code locally
- Staging Area: temporary holding place for all changes before committing
- Local Repository: place for the commited changes locally
- Remote Repository: GitHub server for sharing & backing up code

## Git Commands
- git init: to initialize a repo or re-initialize an existing one
- git add . or git add {{file name}}: to push file(s) from local to Staging, ready for finalization
- git commit -m "feat: message": push the file(s) from Staging to local repo (finalize snapshot of the staged changes)
- git push origin main: send the files to the remote repo
